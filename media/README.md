# 图标文件说明

请将以下图标文件放在这个 `media` 文件夹中：

## 侧边栏图标
- **文件名**：`en.svg`
- **用途**：VS Code左侧活动栏中的插件图标
- **推荐尺寸**：24x24 像素或更大
- **格式**：PNG（支持透明背景）

## 图标要求：
- 颜色：建议使用单色图标，适配VS Code主题
- 背景：透明背景最佳
- 风格：简洁明了的图标设计

## 使用方法：
1. 将 `en.svg` 文件复制到此文件夹
2. 重新编译插件：`npm run compile`
3. 按 F5 测试插件

如果没有提供图标文件，插件仍然可以正常工作，但会显示默认图标。
